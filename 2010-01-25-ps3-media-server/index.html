PS3 Media Server

  • 2010-01-25
    • Tweet

PS3 Media Server es una aplicación JAVA con una interfaz gráfica muy sencilla e intuitiva que permite conectar una playstation 3 a nuestro ordenador como si fuera un servidor uPnP. Esto habilita la opción de compartir archivos en stream para reproducirlos en nuestra PS3 desde el disco duro del ordenador.
El funcionamiento de esta herramienta es sencillísimo. La descargamos desde aquí, la descomprimimos y mientras, encendemos la playstation 3. Nos volvemos al PC y abrimos el servidor tecleando en una terminal, en la carpeta en la que hayamos descomprimido el archivo, la siguiente línea:
sudo sh PMS.sh
Después solo hay que configurarlo un poco. Para seleccionar los archivos compartidos que aparecerán en el XMB de PS3 vamos a la pestaña "Ajustes Navegar/Compartir" y le indicamos qué directorios de nuestro equipo deseamos compartir.
Toda la descripción anterior es para su instalación en un SO basado en Linux pero los desarrolladores de PS3 Media Server también han pensado en los no-geeks que todavía utilizan windows y han compilado una versión que se puede descargar aquí aunque no la he probado y no sé qué tal funciona. Sin embargo puedo afirmar que la versión para Linux se comporta excepcionalmente bien y recomiendo a todos los usuarios de PS3 que la prueben.
