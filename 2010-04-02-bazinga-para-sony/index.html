Bazinga! para sony

  • 2010-04-02
En menos de 48 horas ya he encontrado una buena forma de saltarse la odiosa actualización 3.21 del firmware de PS3. Y es realmente sencillo. Básicamente, la PS3 solicita una web de los servidores de Sony cuando se conecta a playstation network, que contiene la restricción del firmware y demás porquerías. Pero podemos modificar esa restricción que solicita el DNS redireccionándola desde un servidor web adecuado. Hay guías y programas por internet para lograr esto aunque necesitan un ordenador y además con windows y yo no gasto de eso. Pero no hace falta porque podemos utilizar un servidor DNS que no sea nuestro, siempre y cuando esté configurado para ello. Este servidor tendría esa web pero solicitaría la versión 0.00 del firmware en vez de la 3.21 antes de redireccionar. Si la PS3 tiene un firmware inferior al requerido, pedirá la actualización pero si esto no ocurre, podremos conectarnos a playstation network sin actualizar.
En fin, hace un tiempo, a alguien se le ocurrió configurar un servidor DNS de esta forma, así que no necesitamos software para resucitar nuestra cuenta y volver a visitar la store de playstation.
Para conectarnos a playstation network a través de ese servidor solo necesitamos hacer un bypass cambiando el DNS primario por 67.202.81.137 y anulando el secundario, en el menú de configuración de redes de nuestra playstation 3 fat. Y ya está.
Pero ya que estamos hablando de bypasses de DNS, os aconsejo que cambiéis las DNS de vuestro ordenador a:
DNS primaria: 8.8.8.8
DNS secundaria: 8.8.4.4
Las razones, las daré algún día si fuera necesario y creedme si os digo que ojalá no lo sea...

P.S. Bazinga significa "zas! en toda la boca" según el Dr. Sheldon Cooper.
