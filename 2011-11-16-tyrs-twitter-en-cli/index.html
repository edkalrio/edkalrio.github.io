Tyrs. Twitter en CLI

  • 2011-11-16
  • Linux  
  • twitter  
  • Terminal. Shell  

En este blog hemos recomendado twidge como cliente de microblogging para terminal.

Sin embargo, hace unos días, leí un post revelador en muylinux.com sobre un pequeño programa libre, escrito en python cuyo nombre es Tyrs.

Tyrs es de lejos el mejor software para controlar twitter e identi.ca desde el terminal. Twidge y el resto de clientes anteriores a tyrs eran demasiado simples y poco intuitivos, cosa que en un programa de terminal no es raro pero tyrs utiliza ncurses para crear una interfaz a base de unicode como en mocp o mc, programas que también comentamos. Tyrs es ligerísimo, como no puede ser de otra forma. Su proceso de instalación es tan sencillo como escribir en un terminal:

sudo easy_install http://pypi.python.org/packages/2.7/t/tyrs/tyrs-0.5.0-py2.7.egg

Esto instalará el "huevo de pitón" de Tyrs y luego solo quedará configurarlo. El setup básico no lleva más de 2 minutos aunque tiene una potencia de scripting increíble para modificar su aspecto o programar tweets automáticos. Borrad de vuestros ordenadores twidge para dejar sitio al nuevo rey de los terminales.

Mientras escribía este post me ha venido a la mente una duda existencial. Si queremos un cliente que sea intuitivo y vistoso, por qué no usamos un programa gráfico? Si leéis este blog a menudo ya sabréis la respuesta. Porque es más geek usar un programa CLI.
