Ubuntu para Android

  • 2012-02-21
  • Linux  
  • Ubuntu  
  • Android  

"Solo hace falta que cojas el smartphone y lo conectes a un monitor para conseguir una inmersión total en un escritorio Ubuntu con todas las aplicaciones nativas que quieras, ejecutándose en el mismo dispositivo, dentro de Android."

Parece que Canonical quiere revolucionar la industria de las telecomunicaciones de nuevo con una idea tan revolucionaria como utilizar los smartphones de última generación, supervitaminados, con múltiples núcleos y 1GB de RAM o más para algo más productivo y acorde a su potencial. Ubuntu para Android significa que un móvil se puede convertir en la CPU de un ordenador de escritorio con todas las aplicaciones propias de Ubuntu, Unity e incluso la nueva interfaz de Ubuntu TV que transformaría un dispositivo Android en un media center.

Todo esto va a ser posible gracias a Linux. Android y Ubuntu se ejecutarán a la vez sobre el mismo kernel consiguiendo una simbiosis revolucionaria que aunque no tiene nada que ver con la versión de Ubuntu para smartphones (que es un proyecto independiente y también nos apasiona), nos parece increíble. Además, al estar corriendo en el mismo kernel, Ubuntu para Android tendrá acceso a los contactos, calendarios y demás datos de Android. La sincronización es inmediata y automática. Incluso va a ser posible recibir llamadas desde el escritorio de Ubuntu.

La idea de Ubuntu para Android es tan buena que me extrañaría mucho que no la empezasen a plagiar todas las grandes marcas tecnológicas. Sobre todo Microsoft, que le interesaría en especial, fomentar el uso de Windows 8 para ARM.
