Instalar los drivers de Nvidia en Debian

  • 2012-08-26
  • Linux  
  • nvidia  
  • debian  


El otro día os explicamos cómo podíais instalar Wine desde PPA en vuestro sistema Debian.
Una de las cosas que también es importante a la hora de utilizar un sistema linux, sobretodo si vamos a dedicarlo a algún juego, es cómo instalar los drivers propietarios de Nvidia.

Instalar los drivers de Nvidia en Debian 6 puede ser un poco lioso, pero si seguimos las siguientes instrucciones lo conseguiremos a la primera sin mucho trabajo.

Paso 1 - Instalar los módulos correspondientes y configurarlos: para ello ejecutamos en un terminal:


# sudo apt-get install module-assistant nvidia-kernel-common
# sudo module-assistant auto-install nvidia
# sudo apt-get install nvidia-kernel-`uname -r` nvidia-glx
# sudo apt-get install nvidia-settings nvidia-xconfig
# sudo depmod -a  (No tiene que Salir ningun Dato)
# sudo modprobe nvidia (No tiene que Salir ningun Dato)

Paso 2 - Modificar el archivo xorg.conf

Para ello, ejecutamos en un terminal: # sudo gedit /etc/X11/xorg.conf

Buscamos el apartado "Section Device" y sustituimos su contenido por lo siguiente:

Section "Device"

Identifier "Device0"
Driver "nvidia"
VendorName "NVIDIA Corporation"
EndSection

Paso 3 - Reiniciar.

Reiniciamos el equipo, o, con reiniciar gdm debería valer. Para reiniciar gdm tecleamos en un terminal:

sudo /etc/init.d/gdm3 restart

Con estos pasos ya tendremos nuestra distribución debian con los drivers privativos de nvidia que, junto con wine, podremos ejecutar juegos de windows con aceleración 3D sin problemas.
