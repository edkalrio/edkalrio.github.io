Libros para el verano

  • 2017-08-01
  • libros  
Miles de personas comienzan mañana sus vacaciones estivales para disfrutar de un merecido descanso después de un largo año de trabajo.

Se ha convertido en tradición que escriba este post para hacer unas recomendaciones literarias —frikis, por supuesto— para disfrutar durante las vacaciones en un libro electrónico o tablet. Este año, todos los libros que he seleccionado están en inglés porque me he cansado de malas traducciones hechas por filólogos que saben mucho de literatura pero muy poco de informática. A cambio, todos están disponibles gratis, sin las limitaciones que el copyright impone. Desde luego, todos merecen la pena. No puedes perdértelos!

  • A Repository of Unix History and Evolution
  • Automate the Boring Stuff with Python
  • Code Music with Sonic Pi
  • HTTP/2 Explained
  • A Course in Machine Learning
  • A Field Guide to Genetic Programming
  • Project Oberon: The Design of an Operating System, a Compiler, and a Computer
  • AWS Lambda For The Impatient

Feliz verano a todos y gracias por seguirnos.
