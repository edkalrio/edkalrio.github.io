Web como Windows 3.1

  • 2010-01-27


Utilizando JavaScript, XHTML y PHP, se ha conseguido crear una pagina web que simula identicamente un Windows 3.1.
Podemos acceder a MS-Dos, aplicaciones que venian con dicho sispema operativo, un navegador web y jugar un clasico buscaminas.
Aun no se pueden guardar archivos en el servidor para un posterior uso, pero supongo que todo seacuestion de tiempo.
Bienvenido a Windows 3.1, cuando Windows no era mas que un interface de MS-Dos.
