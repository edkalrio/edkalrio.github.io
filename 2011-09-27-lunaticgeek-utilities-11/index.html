Lunaticgeek Utilities 1.1


Hace una semana, en este hilo, os presenté un script que creé con la finalidad de facilitar la instalación de los programas mas elementales para un uso normal de Ubuntu, para no echar nada de menos, y, cuando se instala el sistema de nuevo, no tener que andar instalando las aplicaciones una a una y ahorrarse todo ese rollo.
Hoy os traigo una actualización del script con varios añadidos y modificaciones:

-He añadido el programa qBittorrent
-He actualizado el programa Openshot, añadiéndole efectos, plugins y extras
-He añadido todos los extras de la comunidad de banshee, menos la extensión LIRC
-He añadido la instalación de los extras restringidos de Ubuntu (ubuntu-restricted-extras)
-He añadido la instalación de Wine 1.3

También os quiero presentar el proyecto oficial en Google code:  http://code.google.com/p/lunaticgeek-utilities/

Desde aquí podéis llevar un seguimiento mas de cerca del script, acceder a la sección de descargas, entrar a los Wiki para ver los Changelog, O incluso podéis participar enviando incidencias y sugerencias a la sección de Issues

Para descargar la versión 1.1 de Lunaticgeek Utilities podéis hacerlo de aquí:

 DESCARGAR LUNATICGEEK UTILITIES 1.1

Podéis seguirnos en Twitter en @lunaticgeek y si queréis que añadamos mas programas a este script, no dudéis en comentarlo, que prometemos que lo tendremos en cuenta para futuras actualizaciones.

Quiero sacar una actualización dentro de dos semanas o así, con todo lo que se me haya pasado en esta versión, debido a que el día 13 de octubre sale Ubuntu 11.10 y así, para dicha versión se podrá aplicar el script lo mas depurado posible para ahorrarnos mucho trabajo a la hora de instalar los programas cuando reinstales el sistema que, yo por ejemplo, lo voy a hacer el mismo día que salga.
