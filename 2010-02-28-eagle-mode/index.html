Eagle mode

  • 2010-02-28
Por casualidad descubrí el otro día un programa para Linux que me parece absolutamente revolucionario. Eagle mode es una especie de navegador de archivos extendido que permite leer archivos multimedia, jugar al ajedrez e incluso modificar el kernel de Linux y todo ello a través de una interfaz con zoom fractal aunque con una skin feísima que recuerda más bien a windows 98. Este programa no es reciente pero es un gran desconocido y la verdad es que me parece el navegador más innovador de cuantos he probado. en su web, podréis descargar la versión precompilada en .deb para Ubuntu o la que queráis. Incluso ofrece un ejecutable .exe por si algún usuario de windows quiere probarlo.

Enlace: http://eaglemode.sourceforge.net/index.html
