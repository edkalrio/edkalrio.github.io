World of Goo para todos

  • 2011-09-20
  • #world of goo
    • Tweet

Ya hemos hablado en alguna otra ocasión en este blog de este juego que, en mi opinión, es bastante curioso. Para quienes no lo recordéis, podéis leerlo de nuevo aquí.

Hoy, he decidido facilitaros a todos la descarga del juego, seáis usuarios de Linux, Mac o Windows para que podáis disfrutarlo y paséis tan buenos ratos como yo he pasado con este juego.

Versión de Linux: Descargar

He incluido un .deb para instalarlo como un programa más del equipo y una versión en un .zip que podéis descomprimir donde queráis y, dando permisos a los archivos "WorldOfGoo" y "WorldOfGoo.bin" podáis jugar sin instalar nada más. También he incluido una aplicación llamada "Gootool" que sirve para añadir pantallas personalizadas a World of Goo, junto a una web de recursos para descargar mapas.

Versión de Mac: Descargar

He incluido un .dmg junto a la aplicación "Gootool" y la web de recursos para descargar mapas.

Versión de Windows: Descargar

He incluido un instalador, una entrada de registro para cambiar el idioma al que queramos y la aplicación "Gootool" junto a la web para descargar mapas y recursos.
