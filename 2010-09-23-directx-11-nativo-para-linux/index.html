DirectX 11 nativo para Linux

  • 2010-09-23
  • DirectX  
  • Linux  
Parece que Luca Barbieri, un desarrollador de software ha encontrado la forma de implementar DirectX en Linux para poder ejecutar aplicaciones Direct3D de forma nativa sin necesidad de una conversión previa a OpenGL como viene haciendo wine hasta ahora.
La importancia de este desarrollo es crucial para que los que intentan de una forma u otra jugar videojuegos de windows en sistemas basados en Linux logren los mejores resultados y el más alto rendimiento. Es una gran noticia, además, porque se trata de software libre que podrá ser mejorado por cualquiera con los conocimientos suficientes de programación. Esto nos puede llevar a situaciones muy graciosas como que la librería de DirectX de Linux sea mejor que la desarrollada por microsoft para windows o que podamos ejecutar DirectX11 en Linux pero todavía no podamos hacerlo en windows XP. Otra ocasión para demostrar que el software libre y el open source son mejores alternativas que el software privativo, sobre todo el de microsoft.
