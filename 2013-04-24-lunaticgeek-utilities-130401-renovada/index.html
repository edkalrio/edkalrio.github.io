LUNATICGEEK UTILITIES 13.04.01

Como va siendo ya costumbre con cada lanzamiento de Ubuntu, desde aquí publicamos una nueva versión actualizada de nuestro script Lunaticgeek Utilities 13.04.
En este caso, el script ha supuesto un cambio enorme respecto a anteriores versiones. Viene completamente adaptado para 13.04 así como hemos conseguido (por fin) introducir un script de zenity que nos permitirá de esta manera elegir los programas que queremos y los que no queremos instalar en nuestro equipo.



Podríamos escribir un changelog de esta versión, pero ha sido un cambio tan grande como podría ser actualizar desde Android 2 a Android 5, desde un Kernel de Ubuntu 2.6 a un Kernel Ubuntu 3.8, como de Windows Vista a Windows 8... bueno, en este último caso no hay tanto cambio ;)

Por resumir, el script ahora contiene 100 aplicaciones para instalar, pudiendo elegir cuáles de ellas queremos instalar y cuáles no. También dispone de un log que almacena en /home/usuario que nos mostrará un resumen de las aplicaciones que se han instalado bien y las que se han instalado mal. También al finalizar nos mostrará un resumen de las instalaciones en zenity.
En futuras actualizaciones queremos introducir algún tipo de optimización para el sistema del estilo a borrar archivos temporales, limpieza completa del sistema, etc. Estamos estudiándolo.

El script, como siempre, lo tenéis disponible en Google Code para su descarga.


Aquí tenéis la Wiki con el changelog de todas las versiones: Google Code - Changelog o también podéis enviar sugerencias y errores a la sección de Issues: Google Code - Issues y los intentaremos solucionar lo más rápido posible.

Espero que esta nueva versión os guste igual que las anteriores, o más aún ya que supone un cambio total en todo el diseño y se han añadido varios programas interesantes, y, sobretodo, que os facilite el odioso trabajo que resulta instalar los programas de nuevo tras un formateo o actualización de Ubuntu.

Seguiremos trabajando para tener Lunaticgeek Utilities siempre a punto.
