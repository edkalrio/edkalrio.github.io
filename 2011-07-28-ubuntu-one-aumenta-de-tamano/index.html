Ubuntu One aumenta de tamaño.

  • 2011-07-28
  • Ubuntu  
  • Spotify  

Ubuntu One ha llegado al millón de usuarios registrados.

¿Que haría un software de Microsoft para festejarlo? Algo parecido a Spotify (tu antes molabas "spotty"), seguro que lo ponía de pago o añadiria odiosas restricciones.

Ubuntu One, a cambio, aumenta su capacidad de 2Gb a 5Gb para todos los usuarios gratuitos.

¿Buena noticia, no? Pues aun hay mas, para los usuarios de Ubuntu One Music Streaming, la capacidad aumenta a 20Gb a un precio mensual de 3.99$.

A eso se le llama calidad de servicio.

Chapó por Ubuntu One.
