Windows: Modo Sudo

  • 2011-01-29
  • #sudo

¿Quien nos iba a decir que en windows podemos encontrar un huevo de pascua de semejantes magnitudes?

Este truco es un modo que te crea una carpeta en la cual están agrupadas todas las opciones de configuración de Windows... y cuando digo todas, son todas.

Para llegar a ello es muy fácil:

-Creamos una carpeta, por ejemplo, en el escritorio
-La llamamos: SudoMode.{ED7BA470-8E54-465E-825C-99712043E01C}
-Magia

Hay gente que recomienda no usarlo en Windows Vista 64 bits. Nosotros desde aquí recomendamos directamente no usar Windows Vista.

Windows 7 64 bits tiene modo Sudo sin problemas =D
