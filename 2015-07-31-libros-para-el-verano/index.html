Libros para el Verano

  • 2015-07-31
  • #libros
    • Tweet
Miles de personas comienzan mañana sus vacaciones estivales para disfrutar de un merecido descanso después de un largo año de trabajo.
Se ha convertido en tradición que escriba este post para hacer unas recomendaciones literarias —frikis, por supuesto— para disfrutar durante las vacaciones en un libro electrónico o tablet. Algunos de los que voy a proponer están en inglés, otros tienen copyright, pero todos merecen la pena. Este año, la lista es menos técnica que otros años, con un ensayo filosófico sobre la tecnología y la biografía de Edward Snowden. No puedes perdértelos.

Building a Rural Wireless Mesh Network
NGINX A Practical Guide To High Performance
Beowulf Cluster Computing with Linux
The Question Concerning Technology
Programación en PHP5 nivel básico
Sin Un Lugar Donde Esconderse
The Docker Book

Feliz verano a todos y gracias por seguirnos.
