Chromium empezará a usar la GPU para acelerar la navegación.

  • 2010-09-01
  • #chromium
  • #gpu

Como podemos leer en The Chromium Blog, llevan tiempo optimizando cada vez mas este navegador. Una de las cosas que están probando actualmente es la aceleración utilizando los potentes chips gráficos que existen hoy en día.

Para ello, siguen con la mecánica de crear diferentes procesos a los diferentes trabajos, que en este caso lo denominan "GPU process"
