Terasology, clon open source de Minecraft

  • 2012-05-07
  • open source  
  • juegos  
Nadie sabe muy bien qué es lo que tiene Minecraft que lo hace un juego tan adictivo. 

Pese a su aspecto ultrapixelado que nos recuerda al clásico 3D dot hero y le aporta un matiz muy friki, Markus Persson, el creador de este juego en Java se ha hecho millonario gracias a las impresionantes ventas de un juego tan "cutre" y caro. Probablemente, el precio excesivo de este juego de construcción sin límites sea su talón de aquiles pero gracias a Benjamin Glatzel de movingblocks.net, ya tenemos una versión de Minecraft gratuita y open source llamada Terasology para que podamos aprender de su código, modificarlo, mejorarlo y disfrutarlo gratuitamente. 

El código fuente de Terasology está disponible en github para su descarga. Esta vez habrá que compilarlo ya que se encuentra en una fase temprana de su desarrollo y no hay disponibles paquetes precompilados.
