Cómo instalar Wine en Debian 6


Normalmente, en este blog somos usuarios de Ubuntu, pero nos gusta probar otras distribuciones. El caso es que ayer estaba probando Debian, haciendo pruebas con Lunaticgeek Utilities, cuando llegó el momento de instalar Wine. En la web de Winehq no viene gran cosa sobre como instalarlo en debian, salvo que lo compiles por tu cuenta, cosa que si eres un usuario avanzado está bien, pero no todo el mundo sabe hacerlo, ni tiene tiempo para ponerse con ello.

Después de mucho buscar por la red, no he encontrado absolutamente nada claro y conciso sobre cómo instalar Wine en una distribución Debian, y lo poco que encontré estaba desfasado.

Por ello quiero explicar cómo podemos instalarlo fácilmente en nuestra distribución Debian 6.

Lo primero que debemos hacer es


1- Agregar el repositorio a sources.list, para ello ejecutamos:

sudo gedit /etc/apt/sources.list 

y añadimos al final:

## Wine Squeeze 
deb http://www.lamaresh.net/apt squeeze main

2- actualizamos los repositorios ejecutando:

sudo apt-get update

3- procedemos a instalar Wine desde apt-get

sudo apt-get install wine

Comenzará el proceso de instalación que, él solo, satisfará todas las dependencias necesarias.

Una vez finalizado el proceso ya tendremos Wine instalado en nuestro sistema con Debian. Para comprobar que efectivamente funciona, deberemos ejecutar en un terminal: wine --version, que nos devolverá la versión de Wine instalada en nuestro sistema.

Espero que os sea útil y os ahorre tiempo buscando y leyendo información desfasada.
