LUNATICGEEK UTILITIES 12.04.02



Ha pasado ya mas de un mes, y muchas son las actualizaciones que ha recibido, tanto Ubuntu 12.04 como su software, y software antiguo adaptándose a esta nueva versión.

Por esto, he añadido unos cuantos arreglos a Lunaticgeek Utilities, corrigiendo tanto fallos mios en la primera versión de 12.04 como fallos de incompatibilidad de las ppa y del software con 12.04.

Changelog.
-Corregidos varios errores de scripting, sobretodo en comentarios y de redundancia de "install"
-Revisadas y reparadas las PPA que no eran compatibles con 12.04
-Eliminado Snes
-Reparado lmms-common
-Eliminado amsn por estar desfasado y desactualizado
-Reparado moc
-Añadido Pidgin
-Eliminado Nautilus Gksu
-Añadido Nautilus-actions-extra
-Eliminado tyrs mientras no funcione
-Añadido qtnx
-Añadido wallch
-Añadido Slurm
-Añadido namebench
-Añadido Screenlets
-Añadido Y-PPA manager

Con esto, solo queda anunciaros que ya tenemos Lunaticgeek Utilities funcionando en modo gráfico en alpha1 gracias a eliashl de LinuxZone. La principal característica de la versión con interface gráfica es que podréis elegir que programas queréis instalar y cuales no. Interesante, ¿verdad?
También estamos estudiando como hacer un script para la instalación de videojuegos, ya que no todo es trabajar ;)

Aquí tenéis la Wiki con el changelog de todas las versiones: Google Code - Changelog o también podéis enviar sugerencias y errores a la sección de Issues: Google Code - Issues

Espero que esta nueva versión os guste igual que las anteriores, o mas aun, y, sobretodo, que os facilite el trabajo. Seguimos trabajando para tener Lunaticgeek Utilities siempre a punto.
