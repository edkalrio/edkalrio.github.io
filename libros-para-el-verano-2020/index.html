Libros para el Verano

  • 2020-07-31
  • #libros
    • Tweet

Como todo lo relacionado con este funesto año, las vacaciones también van a caracterizarse por la extrañeza de la nueva normalidad a la que intentamos adaptarnos como podemos. Pero esta no es razón para abandonar nuestras lecturas estivales. Todo lo contrario. El confinamiento voluntario, la renuncia al ocio nocturno, la escasez de viajes y la situación de desempleo son buenos argumentos para aprovechar estos libros más que nunca.

No podía eludir esta oportunidad para hacer unas recomendaciones literarias —frikis, por supuesto— para disfrutar en un libro electrónico o tablet.

Este año he decidido ofrecer una selección variada, basada en mis intereses actuales, pero de la que cualquier informático puede sacar al menos un buen libro para relajarse y olvidarnos por un momento del mundo postapocalíptico en el que vivimos. No puedes perdértelos!

  • Python One-Liners
  • Python Workout
  • Practical R 4
  • JavaScript: The New Toys
  • Pro ASP.NET Core 3, 8th. Edition
  • Serverless Programming Cookbook
  • Unit Testing Principles, Practices, and Patterns
  • The Design of Web APIs
  • Seriously Good Software
  • SAP ABAP Objects

No podía dejar escapar este momento para confirmar que lunaticgeek.com no se va de vacaciones aunque mi actual ritmo de edición es muy pausado y puede que finalmente no escriba nada, seguiré estudiando y documentándome para ofrecer los mejores artículos que sea capaz de publicar. Feliz verano a todos y gracias por seguirnos un año más.
