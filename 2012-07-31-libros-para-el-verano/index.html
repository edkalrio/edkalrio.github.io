Libros para el Verano

  • 2012-07-31
  • #libros
  • #kindle
Miles de personas comienzan mañana sus vacaciones estivales para disfrutar de un merecido descanso después de un largo año de trabajo.
Parece que se ha convertido en tradición que escriba este post para hacer unas recomendaciones literarias —frikis, por supuesto— para disfrutar durante las vacaciones en un libro electrónico o tablet. Algunos de los que voy a proponer están en inglés, otros tienen copyright pero todos merecen la pena:

The C Programming Language 
Steve Jobs
The GNU Hurd Reference Manual
GNU Emacs Manual
Sistemas Operativos, Diesño e Implementación
Manual de desobediencia a la Ley Sinde
Linux from Scratch
La ética del hacker
Internet Hackers y Software libre

No podía dejar escapar este momento para confirmar que lunaticgeek.com no se va de vacaciones aunque como todo el mundo estará descansando, preveo que la cantidad de noticias interesantes que vamos a publicar va a ser mínima. Feliz verano a todos y gracias por seguirnos.
