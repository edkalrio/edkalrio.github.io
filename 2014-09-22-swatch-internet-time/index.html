Swatch Internet Time

  • 2014-09-22
  • #Apple
  • #wikipedia
    • Tweet
Ayer leí una noticia de TechCrunch sobre la negativa del CEO de Swatch a fabricar smartwatches porque no les veía futuro. 

Sobra decir que estoy totalmente de acuerdo con estas declaraciones. En el artículo se mencionaba algo llamado Swatch Internet Time. Tras unos minutos de investigación en Wikipedia, me quedé alucinado con la tremenda frikada que es el Swatch Internet Time. Como no podía ser de otra forma, he escrito este post para divulgar esta extravagante iniciativa que surgió en 1998.

Swatch Internet Time es un sistema de medida del tiempo. En vez de utilizar los segundos como unidad básica, utiliza beats, que define como la milésima parte de un día. 

Otra de las peculiaridades del Swatch Internet Time es que no existen usos horarios. La hora es la misma en todo el mundo. Como referencia para empezar a contar el tiempo se podría haber escogido el meridiano de Greenwich, para que fuera compatible con el standard internacional. Sin embargo, se eligió la opción más chovinista y absurda que nadie se pudiera esperar. Se definió un nuevo meridiano de referencia que pasa por la sede de Swatch en Suiza. Este meridiano BMT es equivalente a UTC+1.

Swatch comercializó durante un breve periodo de tiempo una gama de relojes con el sistema Swatch Internet Time pero ahora son piezas de coleccionista, así que si queremos saber la hora en Swatch Internet Time tenemos que recurrir a uno de los muchos relojes que hay por internet. No voy a enlazar ninguno en particular porque podemos crear uno nosotros mismos, ya que en PHP hay una función específica para este sistema horario, como se ve a continuación.
