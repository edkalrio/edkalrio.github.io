Libros para el Verano

  • 2014-07-31
  • #libros
    • Tweet
Miles de personas comienzan mañana sus vacaciones estivales para disfrutar de un merecido descanso después de un largo año de trabajo.
Se ha convertido en tradición que escriba este post para hacer unas recomendaciones literarias —frikis, por supuesto— para disfrutar durante las vacaciones en un libro electrónico o tablet. Este año, voy a dedicar el post a Andrew Tanembaum que acaba de jubilarse. En las recomendaciones están sus mejores libros, que lejos de ser farragosos volúmenes técnicos, son muy amenos y accesibles. Algunos de los que voy a proponer están en inglés, otros tienen copyright pero todos merecen la pena:

Redes de Computadoras
Sistemas Operativos Modernos
Sistemas Operativos Distribuidos
Operating Systems Design and Implementation

The Swift Programming Language
Código de Máquina para Principiantes
97 cosas que todo programador debería saber
From L3 to seL4. What Have We Learnt in 20 Years of L4 Microkernels?
La introducción no-tan-corta a LATEX 2ε
Building a Rural Wireless Mesh Network

No podía dejar escapar este momento para confirmar que lunaticgeek.com no se va de vacaciones aunque como todo el mundo estará descansando, preveo que la cantidad de noticias interesantes que vamos a publicar va a ser mínima. Feliz verano a todos y gracias por seguirnos.
