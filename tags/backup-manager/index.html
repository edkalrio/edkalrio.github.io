Se anuncia Backup Manager 1.1 con varias mejoras
  • 2010-09-03
  • #PS3
  • #backup manager
  • #Jailbreak
Se anuncia ya la primera actualizacion del Backup Manager, de mano de los creadores de PSJailbreak.Entre otras, las novedades mas destacables son:• Support for all firmware.• Support for NTFS.• Supports Blu-ray and MKV.• Support for PSX and PS2 games.ChipSpain ha dado mas detalles al respecto:• NFTS Support• New version of Backup Manager 1.1 (OK)• Compatible with all PS3 firmware• Compatible with Blu-ray backups, PS2 and PSX.• Auto Updater via your PC.
