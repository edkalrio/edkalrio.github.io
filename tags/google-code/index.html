Lunaticgeek Utilities 1.1 Hace una semana, en este hilo, os presenté un script que creé con la finalidad de facilitar la instalación de los programas mas elementales para un uso normal de Ubuntu, para no echar nada de menos, y, cuando se instala el sistema de nuevo, no tener que andar instalando las aplicaciones una a una y ahorrarse todo ese rollo. Hoy os traigo una actualización del script con varios añadidos y modificaciones:-He añadido el programa qBittorrent-He actualizado el programa Openshot, añadiéndole efectos, plugins y extras-He añadido todos los extras de la comunidad de banshee, menos la extensión LIRC-He añadido la instalación de los extras restringidos de Ubuntu (ubuntu-restricted-extras)-He añadido la instalación de Wine 1.
