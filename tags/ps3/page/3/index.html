Llegan los piratas a PS3?
  • 2010-08-19
  • #pirata
  • #PS3
  • #modchip
Hoy, el mundo geek puede cambiar si el nuevo y primer modchip para PS3 funciona.Se ha hecho oficial hoy mismo la existencia de un dispositivo en forma de dongle USB que presuntamente ofrece la posibilidad de grabar videojuegos de PS3 en discos duros con sistemas de archivos FAT32 para poder jugar nuestras "copias de seguridad" en la consola de sony sin necesidad de tener el juego original. El PS JailBreak, que es como se ha llamado este pequeño modchip cuesta 170$ y se empezará a comercializar sobre el 27 de este mes.
