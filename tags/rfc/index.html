Leer RFCs desde el Terminal
  • 2019-02-06
  • #UNIX
  • #terminal
  • #RFC
Seré breve. Últimamente estoy leyendo bastantes RFCs y siempre acudo a la web de la ietf porque tiene un formato sobresaliente y sin frivolidades, aunque me gustaría una alternativa de emergencia para estudiar offline. El paquete doc-rfc-std de Ubuntu nos trae todo el catálogo al directorio /usr/share/doc/RFC/ para poder consultar desde nuestro emulador de terminal favorito estos documentos, con herramientas UNIX como less. Es demasiado farragoso tener que ejecutar
