Chromium OS Zero

  • 2010-01-14
    • Tweet

Chromium OS Zero es la nueva versión del sistema operativo de Google, en su formato open source, compilado y modificado por Hexxeh para poder ripearlo en un USB.
La mayor novedad de Chromium OS Zero es su menor tamaño de descarga (256MB), además está compilado desde cero con un código más reciente que el utilizado en la versión cherry.
Hexxeh ofrece, en la página de instrucciones para la instalación para Linux, una alternativa para el código


sudo dd if=ChromeOS-Zero.img of=/dev/X bs=4M
Nos propone utilizar un software específico para ripear archivos .img llamado USB-Image Writer que nos facilitará la instalación de Chromium OS en nuestro USB o tarjeta SD.
